    Manual do Usuário — Velocity Editor Engine

    Bem‑vindo ao Velocity Editor Engine v10.3.0 — sua central de criação e publicação de artigos com ciclo completo, do rascunho à página publicada (ou baixada), sem sair do navegador.

    Primeiros Passos

    1. Preencha os campos do formulário — comece pelo título e conteúdo.
    2. O preview ao lado é atualizado automaticamente enquanto você digita.
    3. Escolha como publicar: KV (envio ao Worker), Baixar HTML (download direto) ou Salvar na Pasta (gravação local via File System Access API).

    Publicação via KV (3 Workers)

    O Velocity Article Engine está configurado para três sites/workers distintos:

    • TEOLOGIA — teologia.pautazero.com (todos os eixos de teologia, arqueologia, escatologia, etc.)
    • JUSTIÇA — justica.pautazero.com (todos os eixos de direito, justiça, STF, etc.)
    • GERAL — pautazero.com (todos os demais eixos)

    O worker correto é selecionado automaticamente com base no campo Eixo preenchido.

    Download e Salvamento Local

    Baixar HTML: Faz o download do artigo completo como arquivo .html — funciona em qualquer navegador.

    Selecionar Pasta: Usa a File System Access API (Chrome/Edge) para escolher uma pasta no seu computador. Após selecionada, o botão "Salvar na Pasta" grava o artigo diretamente no disco, permitindo salvar múltiplos artigos em lote sem precisar selecionar a pasta novamente.

    Nota: A API de acesso ao sistema de arquivos requer HTTPS ou localhost e não está disponível no Firefox/Safari. Nesses navegadores, use o botão "Baixar HTML".

    Gerenciador de Nichos (Eixos)

    Na seção "Gerenciador de Nichos" você pode cadastrar e visualizar todos os eixos (nichos) utilizados. Clique em um chip de nicho para preencher automaticamente o campo "Eixo" do formulário. Isso facilita a organização de artigos por categorias como "plantas-e-natureza", "jardinagem", etc.

    Artigos Relacionados

    Adicione até 5 links de artigos relacionados. Eles aparecerão na coluna da imagem no preview e no artigo final. Use caminhos internos como /artigo/plantas-e-natureza/poda-de-roseiras.html.

    Referências

    Utilize o campo "Referências" para inserir notas bibliográficas ao final do artigo. Uma linha fina quase transparente separa visualmente o conteúdo das referências, que são exibidas em tamanho reduzido. URLs são convertidas automaticamente em links clicáveis, sem necessidade de escrever HTML.

    Agendamento

    Use o campo "Agendar (Data e Hora)" com o seletor nativo datetime-local para definir exatamente quando o artigo deve ser publicado. Deixe em branco para usar a data/hora atual.

    Editor em Tela Cheia

    Clique em "Tela Cheia" para abrir o editor expandido com abas Código/Visual. Ideal para edições longas.

    Modo Automático

    Cole texto bruto na seção "Modo Automático" e clique em "Gerar HTML". O sistema detecta títulos, subtítulos e parágrafos automaticamente.

    Tags e SEO

    Preencha as tags separadas por vírgula. Elas são inseridas como meta keywords e exibidas no rodapé do artigo.

    Imagens e Avatar

    Arraste uma imagem para a zona de upload ou cole uma URL. Para o avatar do colunista, selecione uma foto que será redimensionada para 64x64px automaticamente.

    Redimensionamento de Imagens no Corpo

    Novidade v10.2.0: Clique em qualquer imagem inserida no corpo do artigo (figure.body-image) para alternar entre cinco tamanhos: muito pequeno (20%), pequeno (35%), médio (50%), grande (70%) e expandido (100%). Um indicador "🔍 Clique para redimensionar" aparece ao passar o mouse. Isso garante flexibilidade total e redução máxima quando necessário.

    Marca d'água / Referência

    Use o campo "Marca d'água" para inserir um código pessoal de registro (ex: 0002-rosas). Essa referência aparecerá discretamente no rodapé do artigo, funcionando como uma assinatura de origem.
